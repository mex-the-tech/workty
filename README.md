## üéâ Getting Started

After the installation, open workty from your applications or Start Menu. You can start managing your Git worktrees right away. 

### Basic Commands
- **Add a Worktree**: Use the command below to add a new worktree:
  ```
  workty add <path_to_new_worktree>
  ```
- **List Worktrees**: To see all your active worktrees, use:
  ```
  workty list
  ```
- **Switch Worktree**: Change to a different worktree easily:
  ```
  workty switch <worktree_name>
  ```
